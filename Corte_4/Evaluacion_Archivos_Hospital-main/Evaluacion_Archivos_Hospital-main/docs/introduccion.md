# Introducción

El presente proyecto tiene como finalidad evaluar diferentes organizaciones de archivos aplicadas a un sistema hospitalario, utilizando los formatos CSV y JSON para almacenar, procesar y analizar información simulada de pacientes.

En el escenario planteado, un hospital desea modernizar el manejo de información relacionada con registros de pacientes, consultas médicas, signos vitales, historial clínico, medicamentos recetados y citas médicas. Por esta razón, se requiere analizar qué tipo de organización de archivos resulta más conveniente para almacenar y procesar dichos datos.

El proyecto se desarrolla mediante un sistema experimental en Python, el cual permite generar datos simulados, leer archivos CSV y JSON, buscar pacientes, agregar registros, generar reportes, mostrar estadísticas básicas, realizar experimentación y producir gráficas para facilitar la interpretación de los resultados.

La comparación entre CSV y JSON se realiza considerando métricas como tiempo de lectura, tiempo de escritura, tamaño del archivo, facilidad de procesamiento, facilidad de visualización y flexibilidad para almacenar información compleja.

Este trabajo permite aplicar conocimientos de programación, organización de archivos, análisis de datos, visualización de información y toma de decisiones tecnológicas, con el propósito de determinar qué formato resulta más eficiente y útil en un contexto hospitalario.
